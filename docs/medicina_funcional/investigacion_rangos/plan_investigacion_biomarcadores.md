# Plan de Investigación: Rangos de Medicina Funcional para Biomarcadores Metabólicos

## Objetivo
Investigar y documentar los rangos óptimos y aceptables de biomarcadores metabólicos básicos según medicina funcional:
- Glucosa
- Insulina
- HbA1c
- Hemoglobina
- Marcadores de resistencia a la insulina

## Fuentes Objetivo
1. **Institute for Functional Medicine (IFM)** - Fuentes primarias
2. **Journals médicos especializados** - PubMed, Nature, Endocrine Society
3. **Publicaciones de medicina funcional** - Journals específicos
4. **Guías clínicas especializadas** - ADA, Endocrine Society
5. **Revisiones sistemáticas** - Meta-análisis recientes
